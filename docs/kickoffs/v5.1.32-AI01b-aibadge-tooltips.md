## 5. DESIGN RULES

**Tooltip balloon — Treasury spec:**
- Background: `T.navy` (`#12243c`)
- Border: `1px solid rgba(182,135,58,0.5)` (brass at 50% opacity)
- Text color: `T.brassLight` (`#e4c786`)
- Font: `mono` (JetBrains Mono), `fontSize: 9`, `letterSpacing: "0.4px"`, `textTransform: "uppercase"`
- Padding: `"3px 7px"`
- Border-radius: **none** (square, consistent with AiBadge)
- Caret: `borderTop: 4px solid rgba(182,135,58,0.5)`, transparent sides, pointing down
- Position: above the badge, centered, `bottom: calc(100% + 6px)`
- zIndex: 9999
- Show/hide: React state `onMouseEnter` / `onMouseLeave` — no CSS transition needed

**Do not:**
- Add border-radius to the balloon
- Use CSS classes or keyframes (keep inline styles only)
- Change existing `AiBadge` outer layout in a way that shifts surrounding text
