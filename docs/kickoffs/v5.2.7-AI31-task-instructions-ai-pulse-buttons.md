## Section 2 — CONTEXT

**What this session does (two changes, two files):**

**Change A — AssignWorkScreen.jsx: byline badge fix (QA carry-over from AI-29)**  
In `StepCard`, the `AiBadge` on the agent byline is currently nested *inside* the `AgentHoverCard` hover-target span. This means hovering the badge also triggers the hover card, making the badge tooltip unhoverable independently.  
Fix: move `AiBadge` *outside* the hover-target span but keep it in the same parent row.

**Change B — TaskInstructionsScreen.jsx: AI pulse buttons (AI-31)**  
The "Re-run All" and "Update Steps →" buttons each need:
1. A small pulsing dot (brass color, `pdot` animation, 4×4px, `borderRadius:"50%"`) adjacent to the button text
2. An `AiBadge` with label `AI_PAT.TASK_PLANNING`

`AI_PAT` is not currently imported in `TaskInstructionsScreen.jsx` — that import must be added.
