### 2. CONTEXT

**Three bugs to fix + one deprecation:**

**Bug 1 — `/api/extract` returns 500 on every upload (critical)**
`api/extract.js` expects a JSON body: `{ fileData: "<base64>", fileType, fileName }`.
But `extractTextFromFile()` in both PersonnelScreen and TeachScreen sends `multipart/form-data` via `FormData`. Vercel's body parser doesn't parse multipart — `req.body` arrives empty — `Buffer.from(undefined, "base64")` throws — 500. The pipeline has never worked in DeepBench. Fix: use `FileReader` to convert file to base64 and POST JSON.

**Bug 2 — AiBadge invisible on brass buttons**
`<AiBadge />` renders brass-colored text inside brass-background buttons — invisible. Rule (STYLE-GUIDE.md Section 4): badge color must match button label color. On brass buttons (`color: T.navy`), badge color = `T.navy`.

**Bug 3 — AiBadge renders on "✕ Cancel" button state**
The `+ Add Courses` button shows "✕ Cancel" when the sub-view is open. AiBadge must not appear on cancel/dismiss states (STYLE-GUIDE.md Section 4).

**Deprecation — TeachScreen route removed from UI**
`RosterScreen.jsx` has `onAddTraining={a => navigate('/bench/${a.id}/teach')}`. Change to navigate to `/bench/${agentId}?tab=training`. PersonnelScreen reads the `?tab` URL param to open the correct tab on load. TeachScreen.jsx file is kept but becomes unreachable from the UI.

### 4. TASKS

**Task 1 — Fix `extractTextFromFile` in PersonnelScreen.jsx**

Replace the existing `extractTextFromFile` function (which uses FormData) with the base64 JSON version. The function is defined near the top of PersonnelScreen.jsx, above `TrainingTab`.

```js
async function extractTextFromFile(file) {
  try {
    // Convert file to base64 via FileReader — API expects JSON { fileData, fileType, fileName }
    const base64 = await new Promise((resolve, reject) => {
      const reader = new FileReader();
      reader.onload = () => resolve(reader.result.split(",")[1]);
      reader.onerror = reject;
      reader.readAsDataURL(file);
    });
    const res = await fetch("/api/extract", {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({ fileData: base64, fileType: file.type, fileName: file.name }),
    });
    if (!res.ok) {
      const err = await res.json().catch(() => ({}));
      throw new Error(err.error || `Extract failed: ${res.status}`);
    }
    const data = await res.json();
    if (data.error) throw new Error(data.error);
    const text = data.text || "";
    return { text, wordCount: data.wordCount || text.split(/\s+/).filter(Boolean).length, error: null };
  } catch (e) {
    return { text: "", wordCount: 0, error: e.message };
  }
}
```
