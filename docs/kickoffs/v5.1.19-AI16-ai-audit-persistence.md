## 1. Goal

After this session, every AI call is persisted to `ai_activity_log` and the panel hydrates lifetime totals on mount.

### Task 2 ŌĆö Add `hydrateFromSupabase()` to `useAIActivity.js`
**File:** `src/hooks/useAIActivity.js`

New exported async function that reads `ai_activity_log` from Supabase and seeds the in-memory store with historical entries. Called once on panel mount.

```js
export async function hydrateFromSupabase(tenantId = 'global') {
  const { data, error } = await supabase
    .from('ai_activity_log')
    .select('*')
    .eq('tenant_id', tenantId)
    .order('created_at', { ascending: false })
    .limit(500);

  if (error) {
    console.warn('[AI log] Hydration failed:', error.message);
    return;
  }

  // Map DB columns back to in-memory entry shape
  const entries = (data || []).map(row => ({
    id:        row.id,
    type:      row.ai_type,
    model:     row.model || 'claude-haiku-4-5',
    tokens:    row.input_tokens || 0,
    latencyMs: row.latency_ms || 0,
    tier:      row.knowledge_tier || null,
    location:  row.feature || 'ŌĆö',
    agentId:   row.agent_id || null,
    cost:      row.cost_usd ? parseFloat(row.cost_usd) : null,
    ts:        row.created_at,
    _fromDB:   true,  // flag to avoid double-writing on notify
  }));

  // Seed store ŌĆö deduplicate against any same-session entries already present
  const existingIds = new Set(_log.filter(e => e._fromDB).map(e => e.id));
  const newEntries = entries.filter(e => !existingIds.has(e.id));
  _log = [..._log.filter(e => !e._fromDB), ...newEntries].slice(0, 500);
  notify();
}
```
