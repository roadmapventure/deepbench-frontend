## 2. CONTEXT

**Item 1 (QA FAIL — second attempt):**

The AiBadge tooltip still clips in the Chat panel header after S-AI01b-patch. The previous fix (`overflow: "visible"` on the header div) was wrong. Root cause is deeper:

- The tooltip uses `position: "absolute"` — it is clipped by **any ancestor** with `overflow: hidden`, not just the direct parent.
- The outer ChatPanel div (DashboardScreen.jsx line 289) has `overflow: "hidden"` and is 2 levels above the AiBadge.
- Setting `overflow: "visible"` on the header div (line 292) cannot override an ancestor's `overflow: hidden`. It has no effect.

**Correct fix:** Change the tooltip in `AiBadge` (SharedUI.jsx) from `position: "absolute"` to `position: "fixed"` and calculate its screen coordinates dynamically using `getBoundingClientRect()` on mouse enter. `position: fixed` is viewport-relative and escapes every `overflow: hidden` ancestor unconditionally.

Also revert the useless `overflow: "visible"` added to the header div in S-AI01b-patch — it does nothing and should be cleaned up.

**Item 8 new requirement (from QA):**
The "Add New Role Prompt" dashed trigger button in ResumeTab.jsx (line 261) needs an AiBadge with label `"Prompt Build"` — same treatment as "Add New Format" in Playbook tab.

### Task 1 — Fix AiBadge tooltip to use position:fixed (SharedUI.jsx)

Replace the entire `AiBadge` component. Current implementation uses `position:"absolute"` for the tooltip. New implementation uses `useRef` + `getBoundingClientRect()` to calculate fixed position on mouse enter.

**New AiBadge:**
```jsx
// ── AiBadge — AI heartbeat diamond badge for AI-touched elements ─────────────
export const AiBadge = ({ style, label }) => {
  const [show, setShow] = useState(false);
  const [tipPos, setTipPos] = useState({ top: 0, left: 0 });
  const ref = useRef(null);

  const handleMouseEnter = () => {
    if (ref.current) {
      const r = ref.current.getBoundingClientRect();
      setTipPos({ top: r.top - 7, left: r.left + r.width / 2 });
    }
    setShow(true);
  };

  return (
    <span ref={ref}
      style={{position:"relative",display:"inline-flex",alignItems:"center"}}
      onMouseEnter={handleMouseEnter}
      onMouseLeave={() => setShow(false)}>
      {/* FEATURE: AI-01 — AiBadge tooltip label */}
      <span style={{fontFamily:mono,fontSize:8,background:`rgba(182,135,58,0.15)`,border:`1px solid rgba(182,135,58,0.4)`,padding:"1px 5px",color:T.brass,letterSpacing:0.3,flexShrink:0,display:"inline-flex",alignItems:"center",gap:3,...style}}>
        <AIDiamond size="6px" color={T.brass}/> AI
      </span>
      {show && (
        <>
          <span style={{
            position:"fixed",
            top: tipPos.top,
            left: tipPos.left,
            transform:"translate(-50%, -100%)",
            whiteSpace:"nowrap",
            fontFamily:mono,
            fontSize:9,
            letterSpacing:"0.4px",
            textTransform:"uppercase",
            color:T.brassLight,
            background:T.navy,
            border:`1px solid rgba(182,135,58,0.5)`,
            padding:"3px 7px",
            pointerEvents:"none",
            zIndex:9999,
          }}>{label || "AI"}</span>
          <span style={{
            position:"fixed",
            top: tipPos.top,
            left: tipPos.left,
            transform:"translate(-50%, 0)",
            width:0,
            height:0,
            borderLeft:"4px solid transparent",
            borderRight:"4px solid transparent",
            borderTop:`4px solid rgba(182,135,58,0.5)`,
            pointerEvents:"none",
            zIndex:9999,
          }}/>
        </>
      )}
    </span>
  );
};
```

Add `useRef` to the existing `useState` import at the top of SharedUI.jsx:
