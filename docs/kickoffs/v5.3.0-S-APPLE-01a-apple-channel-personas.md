## 2. CONTEXT

The Apple Channel Intelligence build (Market Intelligence tab, `docs/APPLE-AGENT-1-v5-DESIGN.md`) is a 6-agent roster. Every downstream session (S-APPLE-02 through S-APPLE-05, S-MARKET-INTEL-01 through 03) references these agents by name, code, and id — so identity has to exist before any capability, skill profile, or screen work can proceed.

This session adds **identity only**: the six agents become real Bench roster members (agents.js + Supabase `agents` row), exactly like Dan Bingham (S-DAN-02) and the three Editor agents (S-EDITOR-02) shipped before their capabilities existed. No Skill Profiles, no Capabilities, no API routes, no Market Intelligence screen — those are separate, already-queued sessions.

**The 6 personas (locked S-APPLE-01a-design, v5.3.0):**
