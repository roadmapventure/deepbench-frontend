## Section 2 — ROOT CAUSE ANALYSIS

### BUG-01: "Planning agent failed: No steps returned after retry"

**Execution path:**

1. `callPlanningAgent()` in `CreateWorkOrderScreen.jsx` POSTs to `/api/plan` with `action: 'prompt-service'` and `capability_slug: 'cap-pm-01'`
2. `plan.js` calls `assemblePrompt({ capability_slug: 'cap-pm-01', ... })`
3. `assemblePrompt` queries `capability_skill_profiles WHERE capability_slug = 'cap-pm-01'` — returns **0 rows** because `cap-pm-01` does not exist in the database
4. `skillProfiles = []` → `sections = []` → `ai-enrichment` guard fires → returns `system_prompt: ""`
5. `sendRequest` receives empty `system_prompt` — throws (after dad5057 patch: "either system_prompt or sections array required")
6. `plan.js` catches and returns 500 `{ error: "..." }`
7. Frontend `callPlanningAgent` checks `data.content?.steps?.length` → undefined → triggers retry → same failure → returns `{ ok: false, error: 'No steps returned after retry' }`

**Confirmed DB state (queried 2026-06-23):**
- Capability slugs in DB: `data-analyst`, `project-manager`
- `cap-pm-01` does NOT exist
- Michelle's assignment: `agent_id: 'michelle'`, `capability_slug: 'project-manager'` ✅
- `project-manager` has skill profiles: `work-order-decomposition`, `capability-assignment`, `execution-plan`, `planning-behavior`, `capability-registry-knowledge` ✅

**Fix:** Replace `'cap-pm-01'` with `'project-manager'` in all 3 occurrences in `CreateWorkOrderScreen.jsx` (lines 42, 60, 305).

---

### BUG-02: Step cards overflow into left column

**Root cause:** Right column `<div>` had no `minWidth: 0`. In CSS Grid, `1fr` columns do not constrain child overflow without `minWidth: 0` on the grid cell. Step cards with wide content push the column past its allocated boundary, bleeding into the divider and left column.

**Status:** `minWidth: 0` was added to the right column div in commit dad5057. **Verify this is working after deploy.** If step cards still overflow after the capability slug fix allows a plan to load, there is a secondary layout issue to investigate — but do not guess. Test first.

const BASE = 'http://localhost:3000';

async function testPromptServiceWithCorrectSlug() {
  console.log('\nL. prompt-service with capability_slug: project-manager');

        tenant_id: 'global',
        goal: 'Draft a compliance review document for Q3 vendor contracts.',
        deliverable_type: 'document-draft',
      }),
    });

    if (!res.ok) {
      const text = await res.text();
      throw new Error(`HTTP ${res.status}: ${text.slice(0, 200)}`);
    }

    const data = await res.json();

    if (!data.content) throw new Error('content field missing from response');
    if (!Array.isArray(data.content.steps)) throw new Error('content.steps not array');
    if (data.content.steps.length === 0) throw new Error('content.steps empty — pipeline returned no steps');

    const step = data.content.steps[0];
    if (!step.label) throw new Error('step missing label field');
    if (!step.type) throw new Error('step missing type field');

    console.log(`  PROMPT SERVICE (project-manager): PASS`);
    console.log(`  → ${data.content.steps.length} steps returned`);
    console.log(`  → title: "${data.content.title || '(no title)'}"`);
    console.log(`  → step 1: "${step.label}"`);
    return true;
  } catch(e) {
    console.error(`  PROMPT SERVICE (project-manager): FAIL — ${e.message}`);
    return false;
  }
}

async function testSuggestGoalWithCorrectSlug() {
  console.log('\nL. suggest-goal with capability_slug: project-manager');
  try {
    const res = await fetch(`${BASE}/api/plan`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        action: 'suggest-goal',
        agent_id: 'michelle',
        capability_slug: 'project-manager',
        deliverable_label: 'Document Draft',
        deliverable_description: 'Professional report or memo',
        tenant_id: 'global',
      }),
    });

    if (!res.ok) throw new Error(`HTTP ${res.status}`);

    const contentType = res.headers.get('content-type') || '';
    if (!contentType.includes('text/event-stream')) {
      throw new Error(`Expected SSE stream, got: ${contentType}`);
    }

    // Read first chunk to confirm streaming starts
    const reader = res.body.getReader();
    const decoder = new TextDecoder();
    let receivedText = '';
    let attempts = 0;

    while (attempts < 20 && !receivedText) {
      const { done, value } = await reader.read();
      if (done) break;
      const chunk = decoder.decode(value);
      if (chunk.includes('"text"')) receivedText = chunk;
      attempts++;
    }
    reader.cancel();

    if (!receivedText) throw new Error('No text tokens received from suggest-goal stream');
    console.log('  SUGGEST GOAL (project-manager): PASS — stream started');
    return true;
  } catch(e) {
    console.error(`  SUGGEST GOAL (project-manager): FAIL — ${e.message}`);
    return false;
  }
}

  }
}
